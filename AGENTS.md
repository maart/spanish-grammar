# Project Context

Это приложение для изучения испанского языка Испании (castellano). Используется испанский язык Испании (не латиноамериканский).

- Спряжения ориентированы на европейский испанский (voseo не используется, только tú/usted).
- Лексика и переводы соответствуют пиренейскому варианту.
- Грамматические правила (например, leísmo не применяется) следуют стандарту Real Academia Española для Испании.

# Constraints

- **Файл должен оставаться однофайловым артефактом** — `spanish-grammar.html` всегда один, никакого разделения на модули, сборщики или зависимости. Это осознанное решение для удобства передачи (открыть в браузере, отправить по почте, положить на флешку). Весь CSS, JS и данные — внутри одного файла.
