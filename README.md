# Испанский справочник

Автономный HTML-справочник по базовой грамматике испанского языка для русскоязычного пользователя. Проект задуман как быстрый офлайн-справочник, а не как тренажер упражнений: открыть файл, найти глагол, открыть справочник времен, проверить исключения или освежить правило.

## Что внутри

- **Глаголы**: поиск по инфинитиву, форме и русскому переводу, карточка глагола, основные и все времена, подсветка форм, отличающихся от регулярного шаблона.
- **Справочник времен**: логические группы времен, быстрые переходы, образование, признаки времени и исключения по кнопке.
- **Наклонения**: indicativo, subjuntivo, imperativo и condicional с пояснениями, триггерами и примерами.
- **Шпаргалки**: короткие таблицы по местоимениям, предлогам, условным предложениям, косвенной речи и похожим конструкциям.
- **Избранное**: список глаголов для повторения и мини-проверка по случайным карточкам.

## Особенности

- Один файл: `spanish-grammar.html`.
- Работает без сборки, сервера и зависимостей.
- Данные хранятся локально внутри HTML.
- Избранное и тема сохраняются в `localStorage`.
- Подходит для пересылки как один файл.
- Ориентировано на испанский вариант языка, включая `vosotros`.

## Как открыть

Самый простой вариант: открыть `spanish-grammar.html` напрямую в браузере. Можно дважды кликнуть по файлу в Finder или перетащить файл в окно браузера.

На macOS то же самое можно сделать из терминала:

```bash
open spanish-grammar.html
```

Дополнительный вариант: запустить локальный сервер.

```bash
python3 -m http.server 8001
```

После этого открыть:

```text
http://localhost:8001/spanish-grammar.html
```
